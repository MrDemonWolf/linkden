# Installation

You'll clone the repo, install dependencies, set a few environment variables, and start the dev servers. The whole process takes about five minutes.

## Prerequisites

- **Node.js 22+** -- Grab it from [nodejs.org](https://nodejs.org), or use a version manager like `nvm` or `fnm`.
- **pnpm 10+** -- Enable it with Corepack:

```bash
corepack enable && corepack prepare pnpm@latest --activate
```

- **Cloudflare account** -- A free account at [cloudflare.com](https://cloudflare.com) is enough for development.

## Clone and Install

```bash
git clone https://github.com/mrdemonwolf/linkden.git
cd linkden
pnpm install
```

> **Tip:** `pnpm install` handles all apps and packages in the monorepo automatically.

## Set Up Environment Variables

Copy the example file and fill in your values:

```bash
cp .env.example .env
```

At minimum, set these four variables:

| Variable | Description | Example |
|----------|-------------|---------|
| `BETTER_AUTH_SECRET` | Secret key for session signing | Generate with `openssl rand -base64 32` |
| `BETTER_AUTH_URL` | URL of the API server | `http://localhost:3000` |
| `CORS_ORIGIN` | URL of the web frontend | `http://localhost:3001` |
| `NEXT_PUBLIC_SERVER_URL` | Public URL of the API server | `http://localhost:3000` |

See [Configuration](/docs/getting-started/configuration) for a full explanation of each variable.

## Push the Database Schema

```bash
pnpm db:push
```

This creates the database tables LinkDen needs to run.

## Start the Dev Servers

```bash
pnpm dev
```

This launches everything:

- **Web app** -- `http://localhost:3001`
- **API server** -- `http://localhost:3000`
- **Docs** -- `http://localhost:3002`

## Verify It Works

1. Open `http://localhost:3001` -- you should see an empty public page.
2. Check your **server console** for a setup URL that looks like this:
   ```
   Setup URL: http://localhost:3001/admin/setup?token=<unique-token>
   ```
3. Open that URL and complete the 4-step setup wizard (Account, Profile, Theme, Done).
4. You'll land on the block builder -- you're all set!
