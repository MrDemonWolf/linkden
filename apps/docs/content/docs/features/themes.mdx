# Themes

LinkDen ships with 7 built-in themes, each available in dark and light variants (14 total). An auto mode follows the visitor's system preference.

## Available Themes

### Default
A clean, modern design with cyan (`#0FACED`) accents. Medium border radius. Works for any brand.

### Corporate Classic
Professional navy (`#1E3A5F`) palette with subtle border radius. Clean and authoritative.

### Corporate Modern
Teal (`#0D9488`) accents with a contemporary feel. Balanced border radius.

### Hacker Terminal
Green (`#16A34A`) on dark background with no border radius. Monospace-inspired, sharp and technical.

### Neon Cyber
Bold purple (`#7C3AED`) accents with large border radius. Eye-catching and vibrant.

### Furry Pastel
Soft pink (`#EC4899`) palette with fully rounded corners (`1rem`). Warm and approachable.

### Furry Bold
Energetic orange (`#EA580C`) with large border radius. Stands out with high energy.

## Theme CSS Variables

Each theme defines the following CSS custom properties for both light and dark variants:

| Variable | Description |
|----------|-------------|
| `--ld-primary` | Primary accent color |
| `--ld-secondary` | Secondary color |
| `--ld-accent` | Accent color for highlights |
| `--ld-action` | Action/button color |
| `--ld-background` | Page background |
| `--ld-foreground` | Main text color |
| `--ld-card` | Card background |
| `--ld-card-foreground` | Card text color |
| `--ld-border` | Border color |
| `--ld-muted` | Muted background |
| `--ld-muted-foreground` | Muted text color |
| `--ld-radius` | Border radius |

## Selecting a Theme

1. Navigate to `/admin/appearance` in the admin panel.
2. Browse the theme preset picker.
3. Click a theme to select it.

Alternatively, select a theme during the setup wizard at `/admin/setup` (Step 3).

## Color Modes

Each theme supports three modes:

- **Light** -- Forces the light variant regardless of system preference.
- **Dark** -- Forces the dark variant regardless of system preference.
- **System** -- Follows the visitor's operating system preference.

Visitors can toggle between light and dark mode using the theme toggle on the public page.

## Customizing Colors

Beyond the theme presets, you can override individual colors through the appearance settings:

- **Primary color** -- Override `--ld-primary`.
- **Secondary color** -- Override `--ld-secondary`.
- **Accent color** -- Override `--ld-accent`.
- **Background color** -- Override `--ld-background`.

You can also add **custom CSS** for fine-grained control over any theme variable or element.
