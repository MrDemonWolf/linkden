# Settings

The settings page at `/admin/settings` contains all configuration options for your LinkDen instance, organized into 11 sections. Each section has its own Save button.

## 1. Profile

| Field | Description |
|-------|-------------|
| Display Name | Your name or brand shown on the public page |
| Avatar URL | URL to your profile image |
| Bio | A short tagline or description (textarea) |
| Show Verified Badge | Toggle to show a checkmark next to your name |

Saves: `profile_name`, `bio`, `avatar_url`, `verified_badge`.

## 2. SEO

| Field | Description |
|-------|-------------|
| Page Title | The `<title>` tag for your public page |
| Meta Description | Meta description for search engines (textarea) |
| OG Image URL | Open Graph image shown when shared on social media |

Saves: `seo_title`, `seo_description`, `seo_og_image`.

## 3. Contact Form

| Field | Description |
|-------|-------------|
| Enable Contact Form | Toggle on/off |
| Delivery Mode | `Database only`, `Email notification`, or `Database + email` |

When disabled, the contact form block is hidden from the public page.

## 4. CAPTCHA

| Field | Description |
|-------|-------------|
| Provider | `None`, `Cloudflare Turnstile`, or `Google reCAPTCHA` |
| Site Key | Public key (shown when provider is not "none") |
| Secret Key | Server-side key (shown when provider is not "none") |

CAPTCHA keys are stored in the database, not in environment variables.

## 5. Email

| Field | Description |
|-------|-------------|
| Provider | `Resend` or `Cloudflare Email Workers` |
| API Key | Provider API key (password input) |
| From Address | Verified sender email address |

Required for contact form email notifications.

## 6. Branding

| Field | Description |
|-------|-------------|
| Show Whitelabel Footer | Toggle the "Powered by LinkDen" footer |
| Custom Text | Footer text when whitelabel is enabled |
| Custom Link | Footer link URL when whitelabel is enabled |

## 7. vCard

| Field | Description |
|-------|-------------|
| Enable vCard | Toggle the vCard download button on the public page |

See [vCard](/docs/features/vcard) for configuring vCard data fields.

## 8. Apple Wallet

| Field | Description |
|-------|-------------|
| Enable Apple Wallet | Toggle wallet pass generation |
| Team ID | Apple Developer team identifier |
| Pass Type ID | Registered pass type identifier |
| Custom QR URL | Optional custom URL for the QR code on the pass |

See [Apple Wallet](/docs/features/apple-wallet) for detailed setup.

## 9. Export / Import

- **Export** -- Download a complete backup as JSON. Includes blocks, settings, social networks, and contact submissions.
- **Import** -- Upload a JSON file to restore data. Supports `merge` mode.

## 10. About

- Shows the current app version from `version.json`.
- Checks for updates against GitHub releases.
- "Update to X.X.X" button with link to the release page when an update is available.
- Link to the GitHub repository.
