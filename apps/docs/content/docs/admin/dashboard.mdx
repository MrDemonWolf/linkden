# Admin Dashboard

The admin dashboard at `/admin` is your command center for managing your LinkDen instance. It provides a quick overview of your page's performance and shortcuts to common actions.

## Accessing the Dashboard

1. Navigate to `/admin` on your LinkDen instance.
2. If you are not signed in, you will be redirected to the Clerk sign-in page.
3. Sign in with the email address in your Clerk allowlist.
4. You land on the admin dashboard.

## Dashboard Layout

The admin panel uses a macOS-inspired design with a persistent sidebar for navigation and a main content area.

### Sidebar Navigation

The sidebar provides access to all admin sections:

- **Dashboard** -- Overview and quick stats (you are here).
- **Links** -- Manage links and blocks.
- **Appearance** -- Themes, colors, and branding.
- **vCard** -- Digital business card editor.
- **Wallet** -- Apple Wallet pass designer.
- **Analytics** -- Views, clicks, and referrer data.
- **Pages** -- Custom pages (privacy, terms).
- **Contacts** -- Contact form submissions.
- **Settings** -- General settings, export/import, and danger zone.

### Quick Stats

The dashboard header shows key metrics at a glance:

- **Total Views** -- All-time page views.
- **Views Today** -- Page views in the last 24 hours.
- **Total Links** -- Number of active links and blocks.
- **Total Clicks** -- All-time link clicks.

### Recent Activity

A feed of recent events on your page:

- New contact form submissions.
- Recent link clicks with timestamps.
- Configuration changes.

### Quick Actions

Shortcut buttons for common tasks:

- **Add New Link** -- Jump directly to the link creation form.
- **View Public Page** -- Open your public link page in a new tab.
- **Export Data** -- Download a backup of your data.

## Version Check

The dashboard footer shows the current LinkDen version and checks for updates. If a new version is available, a notification banner appears with a link to the [Updating guide](/docs/getting-started/updating).
