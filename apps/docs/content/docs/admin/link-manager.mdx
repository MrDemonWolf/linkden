# Block Builder

The block builder at `/admin/builder` is where you create and organize all the content on your public link page. It supports 5 block types and drag-and-drop reordering with a live phone preview.

## Layout

The builder uses a two-panel layout:

- **Left panel** -- Block list with all your blocks in display order.
- **Right panel** -- Live phone preview (visible on `lg` screens and above). On mobile, a floating action button opens the preview in a modal overlay.

## Block List

Each block row shows:

- **Grip handle** -- Drag to reorder.
- **Block type icon** -- Visual indicator of the block type.
- **Title** -- The block title.
- **URL** -- The destination URL (if applicable).
- **Action buttons** (on hover):
  - **Toggle visibility** -- Eye icon (green when enabled, toggles `isEnabled`).
  - **Edit** -- Pencil icon, opens the edit panel.
  - **Delete** -- Trash icon.

Disabled blocks render at 60% opacity.

## Adding a Block

Click the **Add Block** button to open a dropdown menu listing all 5 block types:

| Type | Description |
|------|-------------|
| **Link** | A clickable button with title and URL |
| **Header** | A text heading with configurable level (h1-h6), alignment, and optional divider |
| **Social Icons** | A row of social media icon links with configurable size, style, and spacing |
| **Embed** | Embed content from YouTube, Spotify, SoundCloud, or a custom URL |
| **Contact Form** | An inline contact form with customizable button text and success message |

## Block Edit Panel

Clicking edit on any block opens a slide-in panel from the right with 4 tabs:

### Content Tab

Fields vary by block type:

- **Link**: Title, URL
- **Embed**: Title, embed type (YouTube, Spotify, SoundCloud, Custom), embed URL
- **Social Icons**: Title, social icons JSON
- **Header** / **Contact Form**: Title

### Style Tab

- Icon name input
- Config JSON textarea for advanced configuration

### Options Tab

Reference text listing common config keys: `noFollow`, `newTab`, `animation`, `thumbnail`.

### Schedule Tab

- Start date/time (`datetime-local` input)
- End date/time (`datetime-local` input)
- **Clear schedule** button

Blocks outside their scheduled window are automatically hidden from the public page.

## Block Configuration

All block types share a base configuration with these style options:

| Option | Description |
|--------|-------------|
| `colorVariant` | Color preset for the block |
| `customBgColor` | Custom background color |
| `customTextColor` | Custom text color |
| `customBorderColor` | Custom border color |
| `borderRadius` | Corner rounding |
| `borderWidth` | Border thickness |
| `shadow` | Drop shadow |
| `animation` | Hover/entrance animation |
| `padding` | Inner spacing |

## Reordering Blocks

Drag any block by its grip handle to reorder. The new order is saved immediately via the `blocks.reorder` mutation.

## Phone Preview

The right panel shows a phone-frame preview (280px wide) with:

- A light/dark mode toggle button.
- Only `isEnabled: true` blocks rendered.
- Type-specific preview rendering:
  - **Link**: Button-style bar
  - **Header**: Centered bold text
  - **Social Icons**: Placeholder circles
  - **Embed**: Gray box with embed type label
  - **Contact Form**: Stacked input placeholders
- Empty state message when no enabled blocks exist.

## Visibility Toggle

Each block has a visibility toggle. Toggling a block off hides it from the public page but keeps it in your builder list. This is useful for:

- Temporarily removing a block without deleting it.
- Preparing blocks in advance and enabling them later.
- Time-limited or seasonal content (or use scheduling instead).
