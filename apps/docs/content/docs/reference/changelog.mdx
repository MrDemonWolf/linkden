## v0.2.0

**March 2026**

### Builder

- Touch + keyboard drag-and-drop via `@dnd-kit` (replaces HTML5 DnD)
- Slide-in edit panel replaces modal dialog — preview stays visible while editing
- On `xl` screens: 3-column layout (blocks | edit panel | preview simultaneously)
- Expandable block picker grid replaces dropdown — type cards with icon + description
- Type-specific accent colors on block rows (colored left bar + icon badge per type)

### New Block Type — vCard

- **Download Contact** block (6th block type)
- Generates a VCard 3.0 `.vcf` file for visitors to save your contact info
- Configurable button text and emoji

### Form Block (renamed from Contact Form)

- Block type renamed from "Contact Form" to "Form"
- Added presets: contact, feedback, RSVP, and more

### Apple Wallet

- Glass-morphism card UI with dark navy default colors
- R2-hosted logo upload
- QR code generation via `qrcode` library
- HIG-compliant pass layout (header → primary → secondary → QR)

### Appearance

- Visual theme preset tiles with color swatches (replaces text list)
- Sticky bottom publish bar — appears only when there are unsaved changes
- Banner upload with preset and custom modes
- Verified badge toggle

### Analytics

- Period selector in charts

### Forms (renamed from Contacts)

- Section renamed from Contacts → Forms
- List + detail split view
- Bulk actions bar (mark read, delete)

### Settings

- Sidebar navigation replaces tabs
- OG image template picker with live preview
- "Powered by LinkDen" branding toggle
- LinkStack import migration wizard

### Infrastructure

- `pnpm ship` — canonical alias for Cloudflare production deploy

## v0.1.0 -- Initial Release

**February 2026**

The first public release of LinkDen.

### Core Platform
- **Block builder** with drag-and-drop reordering and live phone preview
- **5 block types**: Link, Header, Social Icons, Embed, Contact Form
- **100+ social brands** from Simple Icons
- **7 built-in themes** with dark and light variants (14 total)
- **Setup wizard** -- 4-step onboarding (Account, Profile, Theme, Done)

### Admin Panel
- Dashboard with stat cards (views, clicks, unread contacts) and mini bar chart
- Block builder with slide-in edit panel (Content, Style, Options, Schedule tabs)
- Social network management page with search and toggle
- Full analytics dashboard with period selector, charts, top links, referrers, countries
- Contact form submissions management with read/unread tracking
- Settings page with 11 configuration sections
- Version update checker

### Features
- **Better Auth** email/password authentication
- **Apple Wallet** pass configuration
- **vCard** generation and download (VCard 3.0)
- **Contact form** with Cloudflare Turnstile and Google reCAPTCHA support
- **Email notifications** via Resend or Cloudflare Email Workers
- **Analytics** -- page views, link clicks, referrers, country breakdown (privacy-friendly)
- **Block scheduling** with start/end dates
- **Color customization** with CSS custom properties
- **Custom CSS** injection
- **SEO** -- dynamic sitemap, robots.txt, Open Graph meta tags
- **Export/Import** data as JSON (merge or replace modes)
- **Whitelabel** branding with custom footer

### Infrastructure
- **Cloudflare Workers** API server with Hono + tRPC v11
- **Cloudflare D1** (SQLite) database with Drizzle ORM
- **Next.js 16** frontend with React 19 and Tailwind CSS v4
- **Turborepo** monorepo with shared packages (api, auth, db, env, ui, validators, email)
- **Better Auth** session-based authentication
- **Alchemy** infrastructure-as-code for Cloudflare deployment
- **Docker** support for self-hosting
